5. Reporting commands

manitou-mgr [--conf=/path/to/configfile (the same as manitou-mdx)] [--quiet]
[[--print-size ]]

--print-size

Print the total database size, and a subset of the database tables with their sizes in bytes

--quiet

No output unless an error occurs. The default is to output information messages in addition to error messages.
