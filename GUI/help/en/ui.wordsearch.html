4. Global keywords search

By default, each distinct word belonging to the body of messages is put into a global inverted index, so that fast word searches can be performed through the entire database.

Word search is invoked from the search toolbar:

Word search toolbar

When hitting the Enter key or pressing the Find button, all messages whose body contain all words of the search box are retrieved and displayed in a new results page. Also the searched words are highlighted on screen within the body of messages, until the user clears the search box input field.

The search is case insensitive and will not find numbers and words containing only one or two characters, or more than 50. It is also possible to specify words that are not to be indexed (and will be ignored in word search), commonly called stopwords. They have to be put into the non_indexable_words database table.

Searching for exact sentences including space or punctuation is done by enclosing the sentence inside double-quotes. See also the substring search capabilities of the Query selection command.
