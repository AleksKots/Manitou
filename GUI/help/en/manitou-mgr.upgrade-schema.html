3. Upgrading a database schema to a newer version

manitou-mgr --upgrade-schema [--conf=/path/to/configfile (the same as manitou-mdx)] [--quiet]
[--db-name=val] [--db-user=val] [--db-host=val] [--db-port=val]

Options:

--db-name

Database name (manitou by default).

--db-user

Database user that will own a newly created database (manitou by default).

--db-host

Database host name to connect to. By default, connect to a local database (by Unix socket).

--db-port

Database port number to connect to. By default, 5432.

--quiet

No output unless an error occurs. The default is to output information messages in addition to error messages.
